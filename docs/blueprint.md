# **App Name**: Textile Talks

## Core Features:

- Product Display: Showcase textile products with images and descriptions.
- Customization Modal: Allow customers to customize orders and see prices.
- Live Chat Widget: Integrate live chat via Evolution API.
- Order Submission to WhatsApp: Send order details to a designated WhatsApp number.
- AI Assistant for Suggestions: The LLM provides helpful advice and serves as a tool that identifies potential extras to be displayed

## Style Guidelines:

- Primary color: Harvest Gold (#B46408) for a warm, earthy feel.
- Background color: Ecru (#F8F7F5) for a soft, natural backdrop, creating a welcoming atmosphere.
- Accent color: Russet (#8F5518) to complement the primary color, adding depth and sophistication.
- Headline font: 'Playfair', a serif, for headlines, lending an elegant, high-end feel; pair with 'PT Sans'.
- Body font: 'PT Sans', a sans-serif, for body text to guarantee excellent readability.
- Use line icons from Material Symbols Outlined for a clean and modern look.
- Maintain a spacious layout with ample white space to improve readability and overall aesthetics.
